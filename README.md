A service for parsing RSS and Atom feeds.

## Getting started

```
git clone git@github.com:sdepold/feedr.git
cd feedr
npm install
bin/feedr.js
```

You can now run the app and start parsing your feeds:

```
http://localhost:8080/?feedUrl=http://blog.depold.com/rss/
```
